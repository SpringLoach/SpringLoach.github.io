#### 直接打印原始对象

```javascript
const obj = ref({ a: 123 })

console.log(JSON.parse(JSON.stringify(obj.value))) // 会自动忽略 Proxy 的包装
console.log(toRaw(obj.value)) // 提取原始对象
```

```html
<!-- 不会输出到页面 -->
<template>
	<template>123</template>

<template>
	<template v-if="true">123</template>
</template>

</template>
```



#### 字符串修改对应变量

在 `vue3` 中，使用 `setup` 语法定义了变量 ` const a = ref(1) const b = ref(2) `，要根据某个字符串（为` 'a'` 或者 ` 'b'`）去动态修改对应变量的值

```html
<script setup lang="ts">
// =======  变量声明  =======
const leftColumn = ref([])
const rightColumn = ref([])
const columnList = {
    leftColumn,
    rightColumn
}

// =======  主流程  =======
const a = 'leftColumn'
columnList[a].value.push(123)

</script>
```



#### watch监听props

```javascript
// =======  类型声明  =======
const props = defineProps({
    // 滚动距离
    scrollTop: {
        type: Number,
        default: 0
    }
})

// =======  主流程  =======
watch(
    () => props.scrollTop,
    () => {
        checkScroll()
    }
)

// =======  函数声明  =======
function checkScroll() {}
```
