## 命名公约

| 概念               | 要求                                    | 示例                      |
| ------------------ | --------------------------------------- | ------------------------- |
| **React 组件名称** | 大驼峰式                                | `StatusBar`、`SaveButton` |
| **Hook 名称**      | 必须以 `use` 开头，然后紧跟一个大写字母 | `useOnlineStatus()`       |
| **Hook**           | 内部包含 Hook 的函数                    |                           |
| **其他函数 名称**  | 不要以 `use` 开头                       | `getSorted()`             |
| 其他函数           | 内部不包含 Hook 的函数                  |                           |
| 属性命名           | 小驼峰式                                | `className`               |
| 事件命名           | 小驼峰式                                | `onClick`                 |
| 事件处理程序命名   | 名称以 `handle` 开头，后跟事件名称      | `handleClick`             |
