#### 使用高阶数组方法  

索引 | 注意项 | 说明 | 对策  
:- | :- | :- | :-
① | 避免报错 | 不能对非数组类型使用这些方法 | `let arr = wx.getStorageSync('info') || []` 
② | 注意结果 | 对空数组使用，得到的结果可能出乎意料 | `if(!arr.length)`



#### 使用双向绑定

```react
/* 单向绑定 */
<input value="{{ value }}" />

/* 正确的双向绑定 */
<input model:value="{{ value }}" />

/* 错误的双向绑定，无法使用对象属性 */
<input model:value="{{ form.value }}" />
```
