### 常用命令

查看所有安装的 node.js 版本

```elm
nvm ls
```

查看所有可以直接（通过命令行）安装的 node.js 版本

```elm
nvm list available
```

安装 16.15.0 版本的 node.js

```elm
nvm install 16.15.0
```

切换当前使用的 node.js 版本

```elm
nvm use 16.15.0
```

查看当前 node.js 版本

```elm
node -v
```

 卸载 16.15.0 版本的 node.js

```elm
nvm uninstall 16.15.0
```
