### window切软件、页面

| --               | --                                   |
| ---------------- | ------------------------------------ |
| `Window + Tab`   | 折叠所有应用，回到桌面               |
| `Window + Tab`   | 分屏切出所有应用，自行选择激活       |
| `Alt + Tab`      | 切出上一个应用                       |
| `Ctrl + Tab`     | 在浏览器/vscode中切换标签页/激活文件 |
| `Ctrl + Page Up` | 在浏览器/vscode中切换至上一个页签    |
| `Alt + Alt `     | 唤出企业微信                         |
| `Ctrl + Number`  | 在谷歌浏览器快速调整到某个tab页      |
