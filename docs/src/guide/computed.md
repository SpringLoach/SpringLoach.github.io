### 循环遍历数组索引

```javascript
 loopTraversalIndex() {
    const arr = ['a', 'b', 'c', 'd', 'e']
    const arrLength = arr.length
    let current = 0
    setInterval(() => {
        current = (current + 1) % arrLength
    }, 1200)
}
```
